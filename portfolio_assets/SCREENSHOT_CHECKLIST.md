# Screenshot Checklist for Upwork Portfolio

Capture these screenshots with realistic demo data:

1. Hero dashboard view with CVInsight AI branding visible.
2. Job context panel showing role family, must-have and nice-to-have skill inputs.
3. Batch upload section with multiple CV files loaded and parse-status indicators.
4. Auto-generated candidate profile preview cards before running analysis.
5. Leaderboard showing ranked candidates with score bars and dimension labels.
6. Individual candidate card showing covered skills, gaps, and seniority signal.
7. Head-to-head comparison panel with full score component and evidence breakdown.
8. API docs page (`/docs`) showing endpoint reference for technical credibility.

Tip: Use consistent browser zoom and clean desktop background for all captures.
