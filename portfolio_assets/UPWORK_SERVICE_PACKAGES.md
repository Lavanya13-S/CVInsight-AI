# Upwork Service Packages

## Basic - Role-Fit Scoring API

- JD-driven CV analysis API
- Five-dimension scoring (skills, constraints, seniority, proximity)
- Structured evidence fields per candidate
- Up to 2 role profiles
- Delivery: 5-7 days

## Standard - Hiring Intelligence Dashboard

- Everything in Basic
- Browser dashboard with batch CV upload and ranked leaderboard
- PDF, DOCX, and TXT parsing with automatic profile extraction
- Configurable must-have and nice-to-have skill inputs
- Head-to-head candidate comparison view
- Delivery: 10-14 days

## Premium - Full Hiring Intelligence Platform

- Everything in Standard
- Semantic proximity engine and advanced role-family weighting
- Persistent job and candidate history with audit logs
- Shortlist export to CSV and PDF
- ATS-compatible integration endpoints
- Delivery: 2-4 weeks

## Optional Add-ons

- SSO and role-based user management
- Team annotation and candidate notes
- Interview outcome feedback loop
- Domain-specific scoring calibration
